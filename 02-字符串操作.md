# 02-字符串操作

## 常用字符串截取

| A1           | 功能                               | 公式           | 效果 |
| ------------ | ---------------------------------- | -------------- | ---- |
| **ABCD-EFG** | 截取左侧开始的前4个字符            | `=LEFT(A1,4)`  | ABCD |
|              | 截取右侧开始的前3个字符            | `=RIGHT(A1,3)` | EFG  |
|              | 从中间第4个字符开始，截取后3个字符 | `=MID(A1,4,3)` | D-E  |
