# 01-常用快捷键

## 单元格选取

- 快速跳转到数据区域的边界："`Ctrl + 上下左右键`"
- 多个单元格同时选中（每次增加一格）："`Shift + 上下左右键`"
- 选择连续的区域："`Ctrl + Shift + 上下左右键`"
- 选取非连续的区域："`Ctrl + Shift + End`"（右下区域）；"`Ctrl + Shift + Home`"（左上区域）
- 移动区域："`Shift + 鼠标拖动`"



------

## 快捷计算

- 求和（纵向）："`Alt + =`" 
- 快速输入当前日期："`Ctrl + ; `"
- 快速输入当前时间："`Ctrl + Shift + ;`"
- 快速输入相同内容：先选中多个单元格，输入信息后，按"`Ctrl + Enter`"
