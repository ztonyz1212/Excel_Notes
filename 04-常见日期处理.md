# 03-常见日期统计





| A1范例         | 功能           | 公式                                                | 效果       |
| -------------- | -------------- | --------------------------------------------------- | ---------- |
| **`2020/1/1`** | 年/月/日       | `=TEXT(A1,"yyyy/mm/dd")`                            | 2020/01/01 |
|                | 年-月-日       | `=TEXT(A1,"yyyy-mm-dd")`                            | 2020-01-01 |
|                | 年月日         | `=TEXT(A1,"emmdd")`                                 | 20200101   |
|                | 年月           | `=TEXT(A1,"yyyy年m月")`                             | 2020年1月  |
|                | 年             | `=TEXT(A1,"yyyy")`                                  | 2020       |
|                | 年(末两位缩写) | `=TEXT(A1,"yy")`                                    | 20         |

|                | 月             | `=TEXT(A1,"mm")`                                    | 01         |
